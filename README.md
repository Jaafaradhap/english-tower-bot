# English Tower Bot 🇬🇧🏰

This Telegram bot sends daily English learning content such as:
- Word of the Day
- Sample Sentence
- Grammar Rule
- Short Reading Text

Built with `python-telegram-bot` and `APScheduler`.

## Setup
- Add your `TELEGRAM_TOKEN` and `CHAT_ID` as environment variables.
- Run `main.py` or deploy to Render.com for free daily delivery.

Made with ❤️ for English learners.
